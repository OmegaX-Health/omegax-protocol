# Examples

This directory contains protocol-compatible examples and research product sets
that are intentionally kept outside the canonical devnet bootstrap and launch
surfaces.

These examples are useful for showing what builders can model on OmegaX, but
they are not part of the shared-devnet release truth by default.

Current examples:

- [Italian Rehab and Full-Cover Research Product](./research-products/italian-rehab-and-fullcover/README.md)
- [Genesis Protect Acute — Claim Simulations](./genesis-protect-acute-claims/)
